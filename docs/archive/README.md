# Documentation Archive

This directory stores historical exploration notes that are useful for traceability but should not be treated as the current methodology or final experiment design.

- `stages/`: previous Stage 2/4 repair notes, workload probes, and tuning records.
- `validation/`: early validation notes and small-scale pilots.

Active claims should be made from the top-level `docs/` files, especially `methodology.md`, `experiment_protocol.md`, and the latest Stage 4 summary.
